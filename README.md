**Kreirati aplikaciju koji će imati korisnički interfejs kao što je dato na slici i sledeću funkcionalnost:**

a)	Pritiskom na dugme Proveri, vrši se provera unetog teksta.

b)	Akcija se obavlja pomoću javne klase Proveri ima: 
-	Konstruktor sa ulaznim parametrom tipa String.
-	Privatni atribut vrednost tipa String.
-	Metoda daLiJeBroj tipa Boolean koja proverava da li je atribut vrednost broj.
-	Metoda brojCifara tipa String koja vraća „BROJ JE JEDNOCIFREN“ ukoliko je jednocifren, „BROJ JE DVOCIFREN“ ukoliko je broj dvocifren i „BROJ JE TROCIFREN“ u koliko je trocifren.
-	Metoda daLiJeZnak tipa Boolean koja proverava da li je atribut vrednost znak.Atribut model koji predstavlja model vozila.
